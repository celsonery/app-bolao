# Free Bolão

Projeto criado para a Copa do Mundo de 2022 no Qatar.

## Instalação das dependências
```bash
yarn
# or
npm install
```

### Iniciando em modo de desenvolvimento.
```bash
quasar dev
```

### Verificando a qualidade da codificação com ESLint.
```bash
yarn lint
# or
npm run lint
```

### Colocando a aplicação em produção.
```bash
quasar build
```
